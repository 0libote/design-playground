# Design Playground

Explore colour palettes, font pairings, and visual directions in live site previews, then export the result as a reusable design pack.

The MVP includes four palettes with custom colour controls, four font pairings, editorial, commerce, and portfolio previews, three motion directions, desktop and mobile preview modes, and ZIP or single-file exports.

Each ZIP contains `DESIGN.md`, CSS tokens, motion CSS, a palette and type board, and three example-site PNGs.

## Run locally

```sh
bun install
bun dev
```

```sh
bun test
bun run lint
bun run build
```
